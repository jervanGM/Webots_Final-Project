Rescue::Rescue ( )

Contructor de la clase Rescue, inicializa todas las variables y habilita todos los sensores.

La velocidad del robot se inicializa en modo FORWARD, que andará recto. Es usado para crear un robot de busqueda

Rescue *mirobot=new Rescue();
