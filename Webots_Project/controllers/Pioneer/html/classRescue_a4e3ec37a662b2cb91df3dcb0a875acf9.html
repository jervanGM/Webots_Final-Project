void Rescue::bug0 ( )

Aplica la tecnica del bug0 en base a una posición objetivo y asigna un valor a deg que se actualiza cada vez que se llama.

Se suele usar en conjunto con controlForward.

bug0();//Da un valor a deg
angle=deg;
controlForward();

La tecnica de bug0 es simple de aplicar, se da un objetivo al robot y este irá a ese objetivo tomando como referencia la dirección hacia el mismo. Mientras va, se deberán usar los sensores para esquivar objetos y usar controlLine() para evitarlos
